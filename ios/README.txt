
## Prepare the build environment

* Device with iOS 12.0 or above

* Xcode 10.0 or above

* Valid Apple Developer ID

* Xcode command-line tools (run xcode-select --install)

* CocoaPods (run bash sudo gem install cocoapods)



## Build and run

1.  First , Extract all project (chocoview.zip)

2.  Run the following instructions in the Project folder
    * pod install

3.  Finally , Open a project file named "chocoview.xcworkspace"
